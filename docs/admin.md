### Create a new user account for a Doctor/Clinic

| Field      | Type | Description |
| ----------- | ----------- | ----------- |
| Doctor name   |Texr input | Doctor/Clinic name |
| Doctor email   | Email input | Doctor/Clinic email |
| Doctor user type   | Dropdown selector | Doctor/Clinic email |
| Brand   | Dropdown selector | Add branding to a Doctor/Clinic |
| Agent   | Dropdown selector | List of agents attached to the selected Brand |
| Doctor product id from Shopify   | Type | Doctor Shopify ID |
| Doctor collection URL in shop.theplanstore.pro  | Type | Shopify collection url |
| Gema   | Dropdown selector | List of Gema types |
| Clinic   | Dropdown selector | List of clinics |
